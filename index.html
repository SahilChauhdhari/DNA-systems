🔬 DNA Data Storage

Convert any text into nucleotide sequences using binary → DNA mapping. Each ASCII character becomes a 4-base codon through 2-bit per base encoding. Includes step-by-step visual pipeline and FASTA export.

🏷 Barcoding System

Generate a unique cryptographic barcode from any DNA sequence. Visualized as a graphical bar pattern with base-composition metadata, GC content, and a full 32-character hash fingerprint.

⚖ Sequence Comparison

Align two DNA sequences using Hamming distance analysis. Matching and mismatching bases are color-highlighted in an alignment view, with similarity percentage and mutation detection.

🌀 DNA Visualization

Interactive strand canvas with hover-to-inspect base pairs, animated double-helix, mutation simulation, and speed control. Each base is color-coded with Watson-Crick pairing information.

📁 File Storage

Encode entire files into DNA sequences. Converts any file to binary and then to synthetic DNA bases. Download the encoded sequence as a FASTA file or upload DNA to recover your file.

🖼 Image Storage

Encode images into DNA sequences. Upload an image, preview it, encode to DNA, and reconstruct the image natively from DNA text in the browser.

Each ASCII character (8 bits) maps to exactly 4 nucleotides. A 10-character string produces a 40-base DNA strand. Complementary base pairs: A↔T (2 H-bonds) and G↔C (3 H-bonds) — the same Watson-Crick rules used in real DNA.
